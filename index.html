Competences

HTML

Fondation invisible du web, il structure le contenu et guide l’affichage des pages.

HTML

HyperText Markup Language (HTML) est un langage standard pour la création de pages web et est ulisé pour :

  • Définir la structure de base d'une page web
  • Créer des hyperliens
  • Intégrer des images et autres médias
  • Créer des formulaires interactifs

CSS

L'art de la mise en page, CSS donne du style au web. Il transforme une structure brute en interface agréable.

CSS

Cascading Style Sheets (CSS) est un langage utilisé pour définir l’apparence des éléments HTML :

  • La séparation du contenu et de la présentation
  • La cohérence visuelle
  • L'adaptabilité
  • La personnalisation de l’expérience utilisateur

JavaScript

Le langage qui donne vie au web. Il rend les pages interactives, dynamiques et réactives.

JS

JavaScript est un langage de l’interactivité web, il rend les pages dynamiques et réactives :

  • Permet les interactions utilisateur
  • Gère les événements et animations
  • Manipule le contenu HTML/CSS
  • Crée des applications dynamiques

WordPress

Le pouvoir de créer votre présence en ligne, sans limites.

WP

WordPress est un système de gestion de contenu, il facilite la création de sites web :

  • Open-source et gratuit
  • Création de sites sans coder
  • Personnalisable avec thèmes et plugins
  • Idéal pour blogs, vitrines ou e-commerce

SQL

Maîtrisez vos données, maîtrisez votre succès.

SQL

Structured Query Language (SQL) est un langage des bases de données, SQL permet de gérer et manipuler les données :

  • Interroger les bases de données
  • Ajouter, modifier et supprimer des données
  • Assurer l’intégrité des informations
  • Communiquer avec le serveur de données

PHP

Le moteur dynamique du web, pour des expériences riches et interactives.

PHP

Hypertext Preprocessor (PHP) est un langage côté serveur, PHP rend les sites dynamiques et interactifs :

  • Génère du contenu dynamique
  • Interagit avec les bases de données
  • Gère les formulaires et sessions
  • Alimente de nombreux CMS comme WordPress
